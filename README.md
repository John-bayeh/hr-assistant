# HR Assistant - Frontend

AI-powered HR Assistant frontend built with React and Vite.

## Features
- Ask HR questions in plain English
- Get instant AI-powered answers
- Ethiopian labor law guidance

## Tech Stack
- React 18
- Vite
- Tailwind CSS
